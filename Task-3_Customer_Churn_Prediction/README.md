# Customer Churn Prediction

## Description
This project predicts customer churn for a subscription-based service using
machine learning techniques.

## Algorithms Used
- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier

## Dataset
Historical customer data including demographics and usage behavior.

## Libraries Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Outcome
The model predicts whether a customer is likely to churn, helping businesses
take proactive retention actions.
